# 🍽️ Zomato Delivery Analytics Dashboard

An interactive analytics dashboard built using **Python, Streamlit, Pandas, and Plotly** to analyze Zomato food delivery data and generate business insights from **42,000+ delivery records**.

---

## 📌 Project Overview

This project focuses on analyzing Zomato delivery operations to understand factors affecting delivery performance such as:

- Delivery time
- Traffic conditions
- Weather impact
- Vehicle performance
- Order type analysis
- Delivery agent efficiency

The dashboard transforms raw delivery data into interactive visual insights to support better operational and business decision-making.

## 🚀 Installation

Clone repository:

```bash
git clone https://github.com/deergh27/zomato-delivery-analytics-dashboard.git
```

Move into project folder:

```bash
cd zomato-delivery-analytics-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Streamlit application:

```bash
streamlit run zomato.py
```
